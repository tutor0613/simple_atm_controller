# simple_atm_controller
Simple ATM controller for coding assessment (take-home project).

### How to clone
```bash
$ git clone https://github.com/tutor0613/simple_atm_controller.git
```

### Build Environment
```bash
g++ : Apple clang version 17.0.0 (clang-1700.0.13.5)
  - Target: arm64-apple-darwin24.6.0
  - Thread model: posix
  - InstalledDir: /Library/Developer/CommandLineTools/usr/bin
```

### How to build & execute demo
```bash
$ cd simple_atm_controller
$ make
$ ./ATM_DEMO
```
